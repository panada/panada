Auto Loader

Feature auto loader memungkinkan Anda untuk me-load class, baik itu library ataupun model secara otomatis tanpa perlu mendeklarasikannya di dalam controller ataupun model.

Konfigurasi

Konfigurasi auto loader terletak pada file apps/config.php di bagian auto_loader.

Untuk menggunakannya, isikan masing-masing nama class yang akan di-autoload-kan ke dalam nilai array seperti contoh berikut.

$CONFIG['auto_loader'] = array('Library_session' ,'Library_db', 'Model_user');
