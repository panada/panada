Library

Library adalah sebuah class yang memiliki tugas spesifik yang fungsinya bisa digunakan oleh controller, model ataupun library lainnya.

Panada telah menyertakan beberapa library yang umumnya paling sering digunakan. Library natif ini terletak pada folder:

panada/library/

Untuk mengetahui fungsi masing-masing library ini dan cara penggunaanya, silahkan lihat di bagian Paket Library.

Membuat Library

Anda juga bisa membuat library sendiri. Library tersebut bisa disimpan di folder:

apps/library/

Yang harus diperhatikan dalam membuat library adalah, setiap nama class harus diawali dengan prefix "Library_".
