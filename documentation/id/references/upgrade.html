Upgrade

Berikut adalah panduan untuk melakukan proses upgrade dari versi sebelumnya ke versi terbaru.

Pastikan Anda sudah membackup data dan aplikasi sebelum melakukan proses backup.

Versi 0.2.1 ke Versi 0.3.1

Siapkan folder temporer dan download versi 0.3.1 di http://github.com/k4ndar/Panada/archives/master kemudian simpan ke folder temporer.

Buat folder baru pada path berikut di aplikasi Panada yang akan diupgrade:

Jika Anda melakukan perubahan pada file-file di bawah ini, silahkan sesuaikan dengan file versi terbaru.

  • Copy semua file dan folder yang ada di panada/drivers/
  • Copy semua file dan folder yang ada di panada/module/
  • Copy file panada/library/active_record.php
  • Copy file panada/library/cache.php
  • Copy file panada/library/module.php
  • Replace file panada/library/db.php
  • Replace file panada/library/session.php
  • Replace file panada/library/tools.php
  • Replace file panada/library/uri.php
  • Replace file panada/gear.php

Perubahan pada file apps/config.php.

Tambahkan bagian berikut di bagian konfigurasi database:

$CONFIG['db']['default']['driver'] = 'mysql';

Sesuaikan dengan nama koneksi dan tipe driver yang Anda gunakan.

Tambahkan setting untuk cache:

$CONFIG['cache']['default']['driver'] = 'default';

Tambahkan setting untuk secret key:

$CONFIG['secret_key'] = '_put_your_random_string_here_';

Sesuaikan konfigurasi session menjadi seperti berikut:

Method-method berikut masih bisa digunakan, namun disarankan untuk menggunakan method yang lebih baru:

  • Library_db::get_results => Library_db::find_all
  • Library_db::get_row => Library_db::find_one
