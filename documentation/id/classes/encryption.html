Encryption

Untuk memudahkan Anda dalam melakukan pertukaran informasi yang sifatnya rahasia, Panada telah menyiapkan sebuah library enkripsi.

Inisialisasi

Sebelum digunakan, library encryption harus diinisialisasi terlebih dahulu. Inisialisasi bisa dilakukan pada konstruktor ataupun langsung di dalam method.

$this->crypt = new Library_encryption();

Berikut adalah contoh penggunaan library encription:

Tipe Encode

Secara default library encryption menggunakan base_64 untuk encoding. Anda juga bisa menggunakan tipe lainnya. Pilihannya adalah: none, base_64 dan hexa_decimal

$this->crypt = new Library_encryption();
$this->crypt->encode_type = 'hexa_decimal';
